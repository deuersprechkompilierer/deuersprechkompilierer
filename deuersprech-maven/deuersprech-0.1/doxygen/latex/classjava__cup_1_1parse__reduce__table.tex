\doxysubsection{Ausführliche Beschreibung}
This class represents the complete \char`\"{}reduce-\/goto\char`\"{} table of the parser. It has one row for each state in the parse machines, and a column for each terminal symbol. Each entry contains a state number to shift to as the last step of a reduce.
