\doxysubsection{Ausführliche Beschreibung}
This class serves as the base class for entries in a parse action table. ~\newline
 Full entries will either be S\+H\+I\+F\+T(state\+\_\+num), R\+E\+D\+U\+C\+E(production), or E\+R\+R\+OR. Objects of this base class will default to E\+R\+R\+OR, while the other two types will be represented by subclasses.
