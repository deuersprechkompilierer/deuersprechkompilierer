\doxysubsection{Ausführliche Beschreibung}
This class represents the complete \char`\"{}action\char`\"{} table of the parser. It has one row for each state in the parse machine, and a column for each terminal symbol. Each entry in the table represents a shift, reduce, or an error. ~\newline
