\doxysubsection{Ausführliche Beschreibung}
This class represents a set of L\+A\+LR items. For purposes of building these sets, items are considered unique only if they have unique cores (i.\+e., ignoring differences in their lookahead sets).

This class provides fairly conventional set oriented operations (union, sub/super-\/set tests, etc.), as well as an L\+A\+LR \char`\"{}closure\char`\"{} operation (see \mbox{\hyperlink{classjava__cup_1_1lalr__item__set_a9e30f462eb3299ee022fe5c9ff01ef71}{compute\+\_\+closure()}}).

Compute the closure of the set using the L\+A\+LR closure rules. Basically for every item of the form\+: 
\begin{DoxyPre}
  [L ::= a *N alpha, l] 
\end{DoxyPre}
 (where N is a a non terminal and alpha is a string of symbols) make sure there are also items of the form\+: 
\begin{DoxyPre}
  [N ::= *beta, first(alpha l)] 
\end{DoxyPre}
 corresponding to each production of N. Items with identical cores but differing lookahead sets are merged by creating a new item with the same core and the union of the lookahead sets (the LA in L\+A\+LR stands for \char`\"{}lookahead merged\char`\"{} and this is where the merger is). This routine assumes that nullability and first sets have been computed for all productions before it is called. \mbox{\Hypertarget{classjava__cup_1_1lalr__item__set_a4d8450eb2a6a28068f8a43395aeca7b3}\label{classjava__cup_1_1lalr__item__set_a4d8450eb2a6a28068f8a43395aeca7b3}}
