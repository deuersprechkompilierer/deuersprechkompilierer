\doxysubsection{Ausführliche Beschreibung}
This class represents a production in the grammar. It contains a L\+HS non terminal, and an array of R\+HS symbols. As various transformations are done on the R\+HS of the production, it may shrink. As a result a separate length is always maintained to indicate how much of the R\+HS array is still valid.

I addition to construction and manipulation operations, productions provide methods for factoring out actions (see \mbox{\hyperlink{classjava__cup_1_1production_af8ea03e66808d6329c52335e50a9f01c}{remove\+\_\+embedded\+\_\+actions()}}), for computing the nullability of the production (i.\+e., can it derive the empty string, see \mbox{\hyperlink{classjava__cup_1_1production_aa63f3050a9865b05fad3746436c3b3ab}{check\+\_\+nullable()}}), and operations for computing its first set (i.\+e., the set of terminals that could appear at the beginning of some string derived from the production, see \mbox{\hyperlink{classjava__cup_1_1production_a3380eb377ae55c2ed71d51df84ec901b}{check\+\_\+first\+\_\+set()}}).

Full constructor. This constructor accepts a L\+HS non terminal, an array of R\+HS parts (including terminals, non terminals, and actions), and a string for a final reduce action. It does several manipulations in the process of creating a production object. After some validity checking it translates labels that appear in actions into code for accessing objects on the runtime parse stack. It them merges adjacent actions if they appear and moves any trailing action into the final reduce actions string. Next it removes any embedded actions by factoring them out with new action productions. ~\newline
 Finally it assigns a unique index to the production.

Factoring out of actions is accomplished by creating new \char`\"{}hidden\char`\"{} non terminals. For example if the production was originally\+: 
\begin{DoxyPre}
  A ::= B \{action\} C D
\end{DoxyPre}
 then it is factored into two productions\+:
\begin{DoxyPre}
  A ::= B X C D
  X ::= \{action\}
\end{DoxyPre}
 (where X is a unique new non terminal). This has the effect of placing all actions at the end where they can be handled as part of a reduce by the parser. \mbox{\Hypertarget{classjava__cup_1_1production_a41a9421462cf5e75f4b4db778df58586}\label{classjava__cup_1_1production_a41a9421462cf5e75f4b4db778df58586}}
