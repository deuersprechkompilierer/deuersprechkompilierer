\doxysubsection{Ausführliche Beschreibung}
This class serves as the main driver for the Java\+Cup system. It accepts user options and coordinates overall control flow. The main flow of control includes the following activities\+: 
\begin{DoxyItemize}
\item Parse user supplied arguments and options. 
\item Open output files. 
\item Parse the specification from standard input. 
\item Check for unused terminals, non-\/terminals, and productions. 
\item Build the state machine, tables, etc. 
\item Output the generated code. 
\item Close output files. 
\item Print a summary if requested. 
\end{DoxyItemize}

Options to the main program include\+: 
\begin{DoxyDescription}
\item[-\/package name ]specify package generated classes go in \mbox{[}default none\mbox{]} 
\item[-\/parser name ]specify parser class name \mbox{[}default \char`\"{}parser\char`\"{}\mbox{]} 
\item[-\/symbols name ]specify name for symbol constant class \mbox{[}default \char`\"{}sym\char`\"{}\mbox{]} 
\item[-\/nonterms ]put non terminals in symbol constant class 
\item[-\/expect \# ]number of conflicts expected/allowed \mbox{[}default 0\mbox{]} 
\item[-\/compact\+\_\+red ]compact tables by defaulting to most frequent reduce 
\item[-\/nowarn ]don\textquotesingle{}t warn about useless productions, etc. 
\item[-\/nosummary ]don\textquotesingle{}t print the usual summary of parse states, etc. 
\item[-\/progress ]print messages to indicate progress of the system 
\item[-\/time ]print time usage summary 
\item[-\/dump\+\_\+grammar ]produce a dump of the symbols and grammar 
\item[-\/dump\+\_\+states ]produce a dump of parse state machine 
\item[-\/dump\+\_\+tables ]produce a dump of the parse tables 
\item[-\/dump ]produce a dump of all of the above 
\item[-\/debug ]turn on debugging messages within Java\+Cup 
\end{DoxyDescription}
